# 📚 Library System in C

A simple library management system developed in C, using dynamic memory allocation and basic data structures.

## 🚀 Features

* Administrator:

  * Add new books
  * Delete books
  * List all books

* Client:

  * Register user
  * Login system
  * Borrow books

## 🧠 Concepts Used

* Dynamic memory allocation (`malloc`, `realloc`, `free`)
* Structures (`struct`)
* Pointers
* String manipulation
* Menu-driven system (CLI)
* Basic authentication logic

## 🛠️ Technologies

* Language: C
* Compiler: GCC

## ▶️ How to Run

1. Compile the code:

```bash
gcc main.c -o library
```

2. Run the program:

```bash
./library
```
